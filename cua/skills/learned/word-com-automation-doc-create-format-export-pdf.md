# word-com-automation-doc-create-format-export-pdf

Create, format, save, and export a Word document to PDF on the desktop client by driving Word via an ASCII-only PowerShell COM script, bypassing IME/typing pitfalls and verifying results through a status file and document re-read.

## Prerequisites

Microsoft Word desktop client installed and activatable on Windows; PowerShell available with permission to run scripts (use -ExecutionPolicy Bypass); a working directory on local disk for the script, content file, status log, and output .docx/.pdf; clipboard/paste capability to enter commands reliably when a non-English IME is active.

## Steps

1. Launch the Microsoft Word client so it is installed and licensed (Start screen is enough; the script will control Word via COM).
2. Write the document content (e.g., article text under any length limit) to a plain text file on disk to avoid typing non-ASCII text through the UI.
3. Write an ASCII-only PowerShell script that uses Word COM automation: CreateObject/New-Object Word.Application, add a Document, insert title and body/numbered list, apply formatting (Heading style, centered bold title font/size, body font/size, automatic list numbering), SaveAs .docx, ExportAsFixedFormat to PDF, then Quit.
4. Have the script wrap everything in try/catch with $ErrorActionPreference='Stop' and write SUCCESS or the error message to a status.txt log file for headless verification.
5. Run the script (e.g., via run_command or Win+R). If using Win+R on a system with a Chinese IME, do NOT type the command directly — click into the field, Ctrl+A to select existing text, paste the command from the clipboard, then press Enter.
6. Wait for execution, then read status.txt to confirm SUCCESS; retry or inspect the error if it failed.
7. Verify visually with a screenshot: document open in Word, formatted title and numbered list visible, character count in the status bar matches expectations, title bar shows the saved file.
8. Verify the exported PDF by reading it back with a document-extraction tool to confirm it contains the correct title and full body content.
