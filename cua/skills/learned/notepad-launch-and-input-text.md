# notepad-launch-and-input-text

Launch Notepad, focus its window, and reliably insert text via clipboard paste with verification

## Prerequisites

Windows environment with Notepad available on PATH; clipboard access for paste_text; window title varies by system locale, so confirm the exact title via list_windows before focusing

## Steps

1. Launch Notepad with run_command('notepad')
2. Wait ~2 seconds for the window to appear
3. Confirm the window exists with list_windows (title like '无标题 - Notepad' or 'Untitled - Notepad' depending on locale)
4. Focus the Notepad window with focus_window using its exact title
5. Insert the desired text with paste_text (clipboard paste is faster and more reliable than simulated keystrokes)
6. Verify success: window/tab title reflects the content and status bar shows expected cursor position (行 1, 列 N) and character count

## Tools Used

run_command, wait, list_windows, focus_window, paste_text
