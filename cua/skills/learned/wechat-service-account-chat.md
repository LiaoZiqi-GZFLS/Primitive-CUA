# wechat-service-account-chat

Open chat with a WeChat service account and send a text message

## Prerequisites

WeChat desktop client is open and the target service account is visible in search results or recent chats; user is on the service account profile page

## Steps

1. Focus the WeChat window
2. Click the target service account in search results or list
3. Click the 私信 (Private Message) button to open chat interface
4. Click the keyboard icon at bottom right to switch to text input mode if needed
5. Click the text input area
6. Type or paste the desired message

focus_window, set_mouse, click, paste_text, uia_inspect, screenshot
