# open-app-via-run-and-type

Open any Windows application using the Run dialog (Win+R) and then input specified text into it.

## Prerequisites

User must be on Windows desktop with keyboard access; target application must be in system PATH or known executable name

## Steps

1. Press Win+R to open the Run dialog
2. Type the application executable name (e.g., 'notepad', 'calc') in the Run dialog
3. Press Enter to launch the application
4. Use paste_text or type_keys to input the desired text into the application window
