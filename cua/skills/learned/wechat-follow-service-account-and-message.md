# wechat-follow-service-account-and-message

Open the WeChat PC client, search for an official/service account, follow it, and send it a text private message

## Prerequisites

WeChat PC client is installed with a desktop shortcut, the account is logged in (or login only requires clicking '进入微信'), the target account name is known, and the message text to send is prepared

## Steps

1. Double-click the WeChat desktop icon to launch the client
2. Click the '进入微信' (Enter WeChat) button to log in, waiting for the main window to load
3. Click the search box in the top-left corner of the main window and paste the account name (e.g., '火眼审阅')
4. Press Enter to open the '搜一搜' (Search) results window and wait for results to load
5. Take a screenshot to identify the result entry whose type is '服务号' (service account) under the account section, then click it
6. On the account profile page, click the green '关注' (Follow) button and verify it changes to the followed state
7. Click the '私信' (Private Message) button to open the conversation window
8. In the chat window, click the keyboard icon at the right end of the bottom toolbar to switch from voice to text input mode
9. Click the message input box to focus it, then paste the prepared text message with paste_text
10. Click the green '发送' (Send) button and confirm success by verifying the message appears as a green bubble in the chat history

## Tools Used

set_mouse, click, type_keys, paste_text, screenshot, wait, magnifier
