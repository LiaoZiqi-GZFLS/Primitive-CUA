# boss-zhipin-job-market-report

Scrape multi-page job listings from BOSS直聘 via infinite scroll, analyze the data across salary/experience/education/location dimensions, and export a PDF report

## Prerequisites

A phone with the BOSS直聘 app (or WeChat) available to scan the QR login code; Python environment with pip for installing fpdf2; target city and job keyword decided in advance

## Steps

1. Open BOSS直聘 (zhipin.com), switch to the target city, and search the job keyword
2. Complete QR-code login to unmask salaries and unlock the full job list
3. If URL page parameters don't work for pagination, use repeated web_scroll to advance through the infinite-scroll list
4. After each screen of scrolling, record the visible jobs with note (company, salary, experience, education, district), labeling entries by page number
5. Continue scroll-and-note until the target number of pages/jobs is collected (~26 jobs per page, ~130 for 5 pages)
6. Aggregate the noted data and analyze across dimensions: salary distribution, experience requirements, education requirements, district distribution, in-demand skills
7. Install fpdf2 via shell if PDF generation fails or library is missing
8. Generate the PDF report with generate_pdf, saving it to a Documents/reports folder with a descriptive filename
9. Save reusable scraping techniques (login requirement, infinite-scroll pagination, masked salary workaround) to long-term memory
