# wechat-pc-moments-like-first-post

在微信PC版朋友圈信息流中给第一条动态点赞（自绘UI需用坐标操作，点赞入口藏在动态的'···'菜单里）

## Prerequisites

微信PC版已安装且可登录；朋友圈信息流中至少存在一条动态；因微信为自绘UI（UIA不可用），所有操作需通过屏幕坐标点击完成，可能需要先探索界面定位按钮位置

## Steps

1. 双击桌面微信图标启动微信，点击'进入微信'完成登录
2. 点击主窗口左侧边栏的圆形朋友圈图标，打开独立的朋友圈窗口
3. 确认当前处于朋友圈信息流（时间线）视图，而非个人相册视图——相册视图没有点赞入口，若在相册视图需返回信息流
4. 定位信息流第一条动态，点击其右下角的'···'按钮，弹出'赞/评论'选项菜单
5. 点击弹出菜单中的'赞'
6. 验证动态下方出现点赞栏（显示'♡ + 自己的昵称'），确认点赞成功
